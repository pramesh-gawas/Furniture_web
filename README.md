## Setup Steps
##backend

- clone using:  git clone https://github.com/pramesh-gawas/furniture_backend.git
```
$ cd furniture_backend
```
- Install Dependency 
```
npm install
```
- create .env file inside furniture_backend and this field 
- PORT =3000
- DATABASE_URL=
- SECREATE_KEY =
- APIURL=http://localhost:5173
- BLOB_READ_WRITE_TOKEN=
- STRIPE_SECRET_KEY =
  
- Start LocalHost Server
```
nodemon index.js  or node index.js
```

##frontend

- clone frontend using:  git clone https://github.com/pramesh-gawas/Furniture_web.git
```
$ cd Furniture_web
```
- Install Dependency 
```
npm install
```
- create .env file inside Furniture_web and this field
  
-VITE_API_URL=http://localhost:3000

-VITE_API_URL=

-VITE_STRIPE_PUBLISHABLE_KEY =

  
- Start LocalHost Server
```
npm run dev
```
